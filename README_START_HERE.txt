Pace Graphs v15 CLEAN - Start Here

1. Unzip the whole folder to:
   C:\Projects\Pace Graphs

2. Double-click:
   run_pace_graphs_v15_WINDOWS_FIXED.bat

3. Wait for the black window to install packages and start Streamlit.
   Leave the black window open.

4. Open/use:
   http://localhost:8515

Important:
- Do not use old localhost:8501 tabs.
- The correct app header/sidebar should show v15 CLEAN.
- The old Metrics strip, Match confidence box, and Include latest partial week control should not appear.

If nothing opens:
- Look at the black window; it should now stay open and show the error.
- Confirm Python is installed and available from Command Prompt.
